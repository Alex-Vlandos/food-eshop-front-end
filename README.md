# Food Ordering Website (Frontend)

A simple food ordering interface built with vanilla JavaScript.  
Users can browse a sample menu, add items to their cart, and fill out a form to submit their order.

## Features
- Display a dynamic food menu
- Add/remove items from the cart
- Input validation for the order form
- Responsive design (works on mobile & desktop)

## Technologies Used
- HTML5
- CSS3 (Bootstrap/Flexbox/Grid)
- JavaScript (ES6)

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
